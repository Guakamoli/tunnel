# tunnel

简单的内网穿透

## 使用

联系管理员([flxxyz](https://github.com/flxxyz)) 开通权限

```shell
# 首先将管理员创建的用户标记写入到本地配置文件中
echo '唯一的用户标记' > ~/.tunnel
```

```shell
# 执行命令自动完成安装
curl https://get.tunnel.paiya.app | sh
```

> shell自动安装脚本可以看这里 -> [Guakamoli/get.tunnel.paiya.app](https://github.com/Guakamoli/get.tunnel.paiya.app)
